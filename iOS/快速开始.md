# 快速开始指南

## 第一步：获取API Key

### DeepSeek
1. 访问 https://platform.deepseek.com
2. 注册/登录账号
3. 进入API Keys页面
4. 创建新的API Key
5. 复制保存API Key

### 小米MiMo
1. 访问小米AI开放平台
2. 注册开发者账号
3. 获取API Key

## 第二步：打开项目

```bash
cd "E:\AI agent project\project2\ZUJIAN\TokenMonitor"
open TokenMonitor.xcodeproj
```

## 第三步：配置项目

1. 在Xcode中选择项目文件
2. 选择你的开发者Team
3. 修改Bundle Identifier（例如：com.yourname.tokenmonitor）

## 第四步：运行应用

1. 选择iPhone模拟器或真机
2. 按 `Cmd + R` 运行
3. 点击右上角齿轮图标进入设置
4. 输入你的API Key

## 第五步：添加小组件

1. 返回主屏幕
2. 长按空白处进入编辑模式
3. 点击左上角 "+" 按钮
4. 搜索 "Token监控"
5. 选择小组件尺寸：
   - 小尺寸：仅显示余额
   - 中尺寸：显示平台和余额
   - 大尺寸：显示完整信息
6. 点击 "添加小组件"

## 小组件功能

- 自动刷新：每30分钟自动更新
- 实时显示：当前可用余额
- 状态指示：绿色正常，红色余额不足
- 多平台支持：可切换显示不同平台

## 常见问题

### Q: 小组件显示"未配置"怎么办？
A: 需要先在App中配置API Key，然后等待小组件自动刷新（最多30分钟）

### Q: 如何手动刷新小组件？
A: 打开App会触发小组件更新

### Q: API Key安全吗？
A: 是的，API Key存储在iOS Keychain中，非常安全

### Q: 支持哪些iOS版本？
A: 需要iOS 16.0或更高版本
